# Flexbox

This repository contains some of the Flexbox layouts and small UI components I built while learning CSS Flexbox.

I created these mini projects to better understand how Flexbox works in real layouts — things like alignment, spacing, wrapping, positioning, and responsive structure. Instead of only learning the theory, I wanted to practice by building small layouts from scratch.

---

## What’s Inside

### Flex Playground

A small practice layout where I experimented with different Flexbox properties like:

* `flex-direction`
* `justify-content`
* `align-items`
* `flex-grow`
* `flex-shrink`

---

### Dial Box Layout

A simple box layout made using Flexbox.

While building this, I practiced:

* wrapping elements
* spacing items evenly
* arranging content inside a container

---

### Navigation Bar

A basic navigation bar built using Flexbox for horizontal alignment and spacing.

This helped me understand how Flexbox is commonly used in real websites for navigation menus.
